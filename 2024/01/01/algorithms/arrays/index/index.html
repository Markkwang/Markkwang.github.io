Arrays

|Post Views:

Notes

Array indices start from 0.
The memory addresses of array elements are contiguous.

In C++, two-dimensional arrays are stored in contiguous memory, so the addresses form a continuous line.

Java does not have pointers and does not expose the addresses of its elements to the programmer. Addressing operations are entirely managed by the virtual machine.
